Override checks

Type something. Below the form is an info field that shows the kind of typed characters.

The uppercase field updates while typing and turns to green on 4 (or more ... instead of the first).
The char count turns to green on 6 (or more ... instead of 8).

password-field:

Explaination

Let's override definitions for the checks with an options object.
In the options use the subkey "checks".

- lcase - disable lowercase (not recommended - it is just for the demo)
- ucase - set required chars to 4 (default is 1)
- count - set password length to 6 (default is 8)
